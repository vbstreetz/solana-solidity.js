# Introduction

Solana-Solidity.js (opens new window) is a JavaScript Client that can be used to deploy and interact with Solang-compiled Solidity contracts. The Solang Compiler (opens new window) can compile Solidity contracts to native Solana BPF contracts. Attempts to emulate ethers.js (opens new window) and anchor.js (opens new window) APIs.

Features:

  • Upload a BPF program
  • Interact with an existing on-chain program
  • Call a constructor of a solidity contract
  • Call a function of a solidity contract
  • Properly decode errors
  • Subscribe to events
